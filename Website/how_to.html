How-To

To use our application first select the application tab (or right click and open in a new tab so that you can still view this page).

The left side of the screen houses the map with street layers, sattelite views, as well as actual pinpoint Divvy locations that can be outlined by their community areas. Highlight over a pinned station to see it's address and information.

The right side houses our data analysis and graphs. Press one of the buttons to have the script generate a graph using D3 to visualize the given data about the Divvy bike share program.
